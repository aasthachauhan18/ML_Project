About This Project

How we built this fuel efficiency prediction system.

How We Completed the Project

1. Data Collection

We used the Auto MPG dataset for training the model.

2. Data Preprocessing

We cleaned the dataset, selected useful features, and prepared it for training.

3. Model Training

We trained a regression model because MPG is a continuous numerical value.

4. Model Saving

We saved the trained model and scaler using pickle files.

5. Backend Development

We used Flask to connect the trained ML model with the frontend.

6. Frontend Design

We built attractive web pages using HTML, CSS, JavaScript, and chart visualization.

Technologies Used

🐍 Python
🧠 Machine Learning
🌐 Flask
🎨 HTML & CSS
⚙️ JavaScript
📊 Chart.js

Project Outcome

This project helped us understand how to build a complete machine learning application by combining data preprocessing, model training, backend integration, and frontend development into one real-world system.

Built with Python, Flask, HTML, CSS, JavaScript, and Machine Learning
